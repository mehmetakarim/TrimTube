# TrimTube
TrimTube, YouTube videolarını URL ile indirip istediğiniz aralığı hızlıca kesmenizi sağlayan bir Electron masaüstü uygulaması. yt-dlp ile indirir, ffmpeg ile keser; videoyu 9:16 dikey Shorts formatına dönüştürebilir ve OpenCV tabanlı yüz takibiyle, sahne değişse bile kişiyi izleyen akıllı kadraj (auto-crop) uygulayabilir.
